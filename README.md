# Ranking VSM — Streamlit z dynamicznymi minimami

## Uruchom lokalnie
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy na Streamlit Community Cloud
1. Wrzuć `app.py` i `requirements.txt` do repo na GitHub.
2. Wejdź na Streamlit Community Cloud.
3. Kliknij **New app**.
4. Wybierz repo, branch i plik `app.py`.
5. Kliknij **Deploy**.
